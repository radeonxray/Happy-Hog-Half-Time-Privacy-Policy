# 1. General
The App does not require user registration and does not offer login functionality.
The App is designed to collect and process minimal data.

# 2. What Data Is Processed
The App does not collect names, email addresses, phone numbers, or other directly identifying personal information.
However, the following may be processed:

### a) Push Notifications – Google Firebase
The App uses Google Firebase Cloud Messaging (FCM) to send push notifications.
If users enable notifications, the following may be processed:
Device push token (generated by Firebase)
Notification preference status
A push token is a technical device identifier used solely for delivering notifications.
Google may process technical data in accordance with its own Privacy Policy.

### b) Technical Data
The App may automatically collect anonymous technical data such as:
Device type
Operating system version
App version
Crash logs
This data is used solely for functionality, stability, and troubleshooting.

# 3. Purpose of Processing
Data is processed only to:
Send push notifications
Ensure proper App functionality
Improve stability and performance
Troubleshoot technical issues
No data is used for advertising or profiling.

# 4. Legal Basis
Processing is based on:
User consent (for push notifications)
Legitimate interest in maintaining App functionality and security
Users may disable notifications at any time via device settings.

# 5. Third-Party Services
The App uses:
Google Firebase (Google LLC)
Used for push notifications (Firebase Cloud Messaging).
Google may process technical data in accordance with its own Privacy Policy.
The Developer does not sell personal data and does not share data with Den Glade Gris or REKOM.

# 6. Data Retention
Because the App does not create user accounts, no personal profiles are stored.
Push tokens are retained only as long as necessary to provide notifications.
Users may:
Disable notifications
Uninstall the App at any time

# 7. User Rights (GDPR)
Under the General Data Protection Regulation (GDPR), users may have the right to:
Access
Rectification
Erasure
Restriction of processing
Objection
Lodge a complaint with a supervisory authority
Since the App does not store directly identifiable user data, certain rights may be limited in practice.

# 8. Security
Reasonable technical measures are implemented to protect data from unauthorized access or misuse.
However, absolute security in internet communications cannot be guaranteed.

# 9. Changes
This Privacy Policy may be updated from time to time.
The latest version will always be available within the App.
